# 여러 단어 찾기 for Whale

웨일 브라우저의 사이드바 확장 프로그램으로 웹 문서에서 한번에 여러가지의 단어를 각각 다른 색으로 표시합니다.  
단어는 쉼표( , )로 구분하며 여러 단어를 찾을 수 있습니다.

### Features

- 사이드바에서 단어 선택 시 해당 단어의 위치로 이동.
- 깊은 검색 가능. (ex. 네이버 카페)
- 추가로 생성되는 단어도 검색. (ex. 유튜브 댓글)
- 검색 단어 기억.
